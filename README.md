## ðŸš€ Getting Started

### Prerequisites
Ensure you have the Flutter SDK installed. If not, follow the [Official Installation Guide](https://docs.flutter.dev/get-started/install).

### Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone [https://github.com/Pankaj-Singh-Rawat/your-repo-name.git](https://github.com/Pankaj-Singh-Rawat/your-repo-name.git)
   cd your-repo-name
   ```

2. Navigate to a specific project
# To run the Namer App
```text
cd namer_app
```

# To run the Notes App
```text
cd notes_app
```

3. Install Dependencies
   ```text
   flutter pub get
   ```
4. Launch the App
   ```text
   flutter run
   ```
